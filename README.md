# Budget Buddy

Budget Buddy is an Android budgeting app developed in Kotlin using Android Studio. The app is designed to help users manage their personal finances by recording expenses, organising them into categories, setting monthly budget goals, and reviewing spending reports.

## Features

- Register and log in with a username and password
- Create and manage expense categories
- Add expense entries with amount, date, description, and category
- Optionally attach a receipt photo to an expense
- Set a monthly total budget goal
- View expenses within a selected period
- View total spending per category
- Store all data locally in a database
- Display spending reports and graphs
- Use South African Rand (R) as the app currency

## Built With

- Kotlin
- Android Studio
- XML
- SQLite

## Project Goal

The goal of this project is to provide a simple mobile budgeting solution that helps users track spending habits and stay within their financial limits.
